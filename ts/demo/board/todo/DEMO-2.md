Create a Python file `fibonacci.py` in the repo root with a function `fib(n)` that returns the nth Fibonacci number (0-indexed).

Acceptance criteria:
- `fib(0)` returns `0`
- `fib(1)` returns `1`
- `fib(10)` returns `55`
